# Logistics Dashboard

StackBlitz par deploy karne ke liye:
1. GitHub repo mein yeh file daalo
2. StackBlitz mein "Import from GitHub" karo
3. URL copy karo aur bhejo

## Setup
- `index.html` mein `SPREADSHEET_ID` aur `API_KEY` apni daalni hai
- Google Sheet public karo (Share → Anyone with link → Viewer)
